variaveis - var let const / elemento responsavel por armazenar informações para que sejam utilizadas no desenvolvimento do sistema.

Tipos de variaveis:
    - Texto (string) "texto"
    - Inteiro (Int, number) Armazena só números inteiros "10"
    - Decimal (Float, Double, Decimal) "10.1"
    - Boleano (True, False) "True"
    - Lista [Varias informações] 


Lógica de programação

Algoritimo - É uma série de passos realizados para resolver um determinado problema.

Inicio: Como fazer um miojo

ir na cozinha
pegar o miojo
pegar uma panela
colocar água esquentar
colocar o miojo na água quente
temperar e servir

Fim:

Condicionais: IF ELSE / SESIM SENÃO

    (IF)SE tiver miojo
        pega o miojo
    (ELSE)SENÃO tiver miojo
        compra miojo

Rotina - Faz os mesmos passos diariamente

Dia 1: 
    Acordar
    Trocar de roupa
    Lavar o rosto
    Tomar café
    Escovar os dentes 
    arrumar o cabelo
    Sair para trabalhar

Variavel - Guarda informação
Função - Guarda o passo a passo sendo 1 ou mais passos

Algoritimo do miojo:

    (IF)SE tiver miojo
        pega o miojo e faz o miojo
    (ELSE)SENÃO tiver miojo
        compra miojo
